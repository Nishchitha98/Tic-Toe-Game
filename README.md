# 🎮 Tic-Tac-Toe Game

A modern and responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## ✨ Features

- Player X and Player O turns
- Winner detection
- Draw detection
- Winning cells highlight
- Congratulations popup
- Responsive design
- Restart game functionality
- Modern glassmorphism UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎯 Game Rules
Player X starts first
Players take turns placing marks
First player to align 3 marks wins
If all cells are filled without a winner, the game ends in a draw
